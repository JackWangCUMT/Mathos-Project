Integration

Integration of a function can be achieved in two different ways. Simpson's method is being used during integration. You can read more about other methods available here.

Integrate a string expression

Any string expression can be integrated as shown below: 
	# integrate 3x^2 from 0 to 2
	puts int("3x^2",0,2)
See in action

Integrate a lambda expression

If you can express the expression (integrand) as a lambda expression, the int might execute faster than the method above. 
	# integrate 3x^2 from 0 to 2
	puts int(lambda {|x| 3*x**2},0,2)
